# 🎲 Dice Roller for SillyTavern

A Baldur's Gate 3 inspired dice roller extension for SillyTavern that detects AI-generated dice roll requests and provides an interactive dice rolling experience.

## ✨ Features

* 🎲 Animated dice rolling
* 🎲 Physical dice support
* 🤖 AI roll request detection
* 💬 Automatic OOC result insertion
* 🎨 Fully customizable theme
* 🔁 Configurable rerolls
* 📱 Desktop & mobile support
* ⚙️ Configurable extension settings

## 📦 Installation

1. Clone this repository into:

```text
public/scripts/extensions/third-party/
```

2. Restart or reload SillyTavern.
3. Enable the extension in the Extensions menu.

## 🎮 Usage

When the AI requests a dice roll, a notification appears.

You can either:

* 🎲 Roll digitally using the animated dice.
* 🎲 Enter the result from your physical dice.

Depending on your settings, the extension can automatically insert or send the OOC roll result.
