# 🎲 Dice Roller for SillyTavern

A Baldur's Gate 3 inspired dice roller extension for SillyTavern.

## Features

- 🎲 Animated D20 popup
- 🎨 Theme customization
- 🔁 Configurable rerolls
- 🤖 AI roll detection
- 💬 Automatic OOC result insertion
- ⚙️ Fully configurable

## Installation

1. Clone into

public/scripts/extensions/third-party/

2. Reload SillyTavern

3. Enable the extension

## Usage

When the AI asks for a roll, a notification appears.

Click **Roll Dice** to open the popup.
